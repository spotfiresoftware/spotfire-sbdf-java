The files in this directory are sample sbdf files.

1.sbdf contains a table with one row with values of all datatypes. The file also contains table and column metadata for all datatypes.

The other files named 1_typename.sbdf contain data of a particular type.

10001.sbdf and 10001_typename.sbdf contain 10001 rows of data.

100.sbdf and 100_typename.sbdf contain 100rows of data.

0.sbdf and 0_typename.sbdf contain zero rows of data. The files only contain table and column metadata.

no_columns.sbdf and no_columns_typename.sbdf contain no columns (and therefore no rows). The files only contain table metadata.
